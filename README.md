# learn-typescript

- https://github.com/TypeStrong/learn-typescript

- TypeScript 严格模式 `"strict": true`
- 禁止隐式 any `noImplicitAny: true`
- 严格的空检查 `strictNullChecks: true`
- TypeScript 中的类型收窄
  - 变量可以从不太精确的类型转移到更精确的类型，这个过程称为类型收窄。
